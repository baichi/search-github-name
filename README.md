
# React项目开发示例

> 基于React+webpack实现通过搜索github的用户名返回用户仓库及点评的功能

## 技术栈
- [x] [Atom](https://atom.io/)
- [x] [React]()
- [x] [Webpack]()
- [x] [Babel]()
- [x] [ES6]()
- [x] [Postcss]()
- [x] [Bootstrap]()
- [x] [React-router]()
- [x] [axios]()
- [x] [re-base]()

## 安装

```
npm install
```

## 运行

```
npm run dev
```

在浏览器中打开 `http://localhost:8080/`
